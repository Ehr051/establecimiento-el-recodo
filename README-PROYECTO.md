
## 🚀 Inicio Rápido

### Para ver la web localmente:
```bash
# Opción 1: Servidor automático
./start-server.sh

# Opción 2: Abrir directamente
open website-el-recodo.html
```

### Para compartir con otros:
```bash
./start-server.sh
# Te dará la IP local para compartir: http://192.168.x.x:8000
```
